# DSBDA_Practicals
 DSBDA (Data Science and Big Data Analytics) practicals


DSBDA Practicals – SPPU TE
📌 Overview
This repository contains practical assignments for the Data Science and Big Data Analytics (DSBDA) course, part of the Third Year Engineering curriculum (TE) at Savitribai Phule Pune University (SPPU).
The aim is to provide hands-on experience with data preprocessing, visualization, machine learning models, and big data tools.

🛠️ Technologies Used
- Programming Language: Python (Jupyter Notebook / Google Colab)
- Libraries:
- NumPy, Pandas (Data Handling)
- Matplotlib, Seaborn (Visualization)
- Scikit-learn (Machine Learning)
- NLTK / TextBlob (NLP tasks)
- Big Data Tools:
- Hadoop / Spark (where applicable)


DataSet:
Take from www.kaggle.com
